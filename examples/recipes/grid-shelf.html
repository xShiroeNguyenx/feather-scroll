🔲 Grid shelf (nhiều hàng)

Lưới sản phẩm 2 hàng, mỗi lần bấm nhảy đúng một cột màn hình — như khối “Danh mục nổi bật” nhiều dòng.

Kết hợp: preset gridShelfgrid: { rows: 2 } slideBy: 'page'navdrag

Đổi số hàng bằng gridShelf({ rows: 3 }). Grid dùng layout track + flex column-wrap, vẫn chỉ một transform GPU.
