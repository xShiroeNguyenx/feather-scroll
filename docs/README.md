- `examples/playground/` — **chỉnh mọi tham số trực tiếp** + builder slider tự tạo hiệu ứng
- `examples/showcase/` — **105 hiệu ứng chạy đồng thời** theo nhóm
- `examples/recipes/` — widget hoàn chỉnh (testimonials, hero, product, gallery + thumbnails)

```bash
pnpm install
pnpm build
# rồi mở examples/index.html qua một static server
npx serve .
```
